# 🌐 Personal Portfolio Website

这是我的 **个人品牌门户网站（Personal Portfolio）** 的在线简历平台。

设计理念来源于我对 **产品经理角色** 的理解：  
> 在 **数据、技术与用户体验之间寻找优雅的平衡点**。

该网站不仅展示我的 **教育背景、项目经验与技能体系**，也体现了我对 **产品结构设计与用户体验** 的思考。

# 🎨 Design Philosophy

本网站遵循三个设计原则：

**1️⃣ Minimalism（极简主义）**

减少视觉噪音，让信息成为设计的核心。

**2️⃣ Information Aesthetics（信息美学）**

通过结构化布局，让信息更易理解与浏览。

**3️⃣ User Experience（用户体验）**

通过流畅交互与合理的信息架构，提高可用性。
