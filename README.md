├── 首页 Hero         → 一句话介绍 + 核心行动点

├── 实习经历 Experience → 4段实习（百度/梧桐电子/湛腾/天融信）
├── 项目 Projects      → 2个深度学习/后端开发项目
├── 技能 Skills        → 技术栈 + 核心能力雷达 + 语言能力
