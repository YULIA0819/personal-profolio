# 个人品牌门户网站（Personal Portfolio）

这是我的**个人品牌门户网站**，一个融合**极简主义设计**与**信息美学**的在线简历。  
整体设计灵感来源于我对**产品经理岗位**的理解——在**数据与用户体验之间寻找优雅的交汇点**。

---

## 📂 内容结构
├── 首页 Hero         → 一句话介绍 + 核心行动点（Call-to-Action）
├── 关于我 About      → 教育背景 + 个人定位
├── 实习经历 Experience → 4段实习经历（百度 / 梧桐电子 / 湛腾 / 天融信）
├── 项目 Projects     → 2个深度学习 / 后端开发项目
├── 技能 Skills       → 技术栈 + 核心能力雷达 + 语言能力
├── 联系 Contact      → 联系方式 + 模拟表单提交
└── Footer            → 回到顶部导航
---

## 🛠 技术栈

### 1. HTML5
- 使用**语义化标签结构**
- 提升页面结构清晰度与可读性

### 2. CSS3
- **Flexbox / Grid 布局**
- **自定义动画效果**
- **媒体查询实现响应式设计**

### 3. 原生 JavaScript
- **光标追踪效果**
- **滚动监测交互**
- **模拟表单提交功能**

---

## 🎨 设计理念

- **极简主义（Minimalism）**：保持清晰的视觉层级与内容重点  
- **信息美学（Information Aesthetics）**：结构化展示技能与经历  
- **用户体验（UX）导向**：流畅交互与直观导航  

---

## 🚀 项目目的

该网站主要用于：
- 构建**个人品牌门户**
- 展示**在线交互式简历**
- 体现**前端开发能力与产品思维**
